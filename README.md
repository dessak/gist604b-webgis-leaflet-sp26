**Module:** 6 WebGIS & Full-Stack Orchestration  
**University of Arizona**  

## Project Description
This repository uses HTML, CSS, JavaScript, and Leaflet to complete the requirements for Module 6 of the 604B — Open Source GIS class. In this module a leaflet map was created to visualize public transportation in the State of Oregon. Layers include City Boundries, public transportation stops, and a layer with busways and pedestrian paths.

## Tools and Technologies
-QGIS  
-Leaflet  
-Visual Studio Code  

## What I did
-Downloaded Data from Geofrabik  
-Processed in QGIS  
-Produced webmap in VS Code using leaflet
